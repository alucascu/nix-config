# Hostname Reference

Three-tier scheme: `pantheon → realm → host (god)`

## Norse — virtual machines

The nine worlds of Norse cosmology hang from Yggdrasil, the world-tree. Each realm is a distinct
but interconnected domain, making them a natural fit for virtual machines.

*Home of the Aesir gods, seat of Odin's hall Valhalla. Connected to Midgard by the rainbow bridge Bifrost.*

| Name | Domain |
|---|---|
| Odin | Wisdom, war, death, poetry — the Allfather |
| Thor | Thunder, strength, protection of mankind |
| Freya | Love, war, magic, death — leader of the Valkyries |
| Freyr | Fertility, sunshine, rain, prosperity |
| Loki | Trickery, shapeshifting, chaos — agent of change |
| Tyr | Justice, law, heroic glory |
| Baldur | Light, purity, beauty — the beloved god |
| Heimdall | Watchman of the gods, dawn, foresight |
| Njord | Sea, wind, fishing, coastal wealth |
| Skadi | Winter, mountains, hunting, skiing |
| Mimir | Wisdom, memory, the well of knowledge |
| Sigyn | Loyalty, fidelity, compassion |
| Bragi | Poetry, eloquence, music |
| Idun | Youth, spring, keeper of the golden apples |
| Forseti | Justice, reconciliation, mediation |
| Ullr | Archery, skiing, hunting, winter |
| Vidar | Silence, vengeance, the forest |
| Vali | Vengeance, archery, born to avenge Baldur |
| Hodr | Darkness, winter — the blind god |
| Hermod | Speed, messengers, the bold |

## Greek — physical machines

Each realm is a distinct, grounded domain — a natural fit for bare-metal hardware.

## Roman — containers

Roman gods often mirror Greek ones but carry distinct character — more civic, martial, and administrative.
The seven hills of Rome serve as realm names.

### Palatine
*The central and most prestigious hill of Rome, traditional home of Romulus and later the emperors.*

| Name | Domain |
|---|---|
| Jupiter | Sky, thunder, kingship, law — king of the gods |
| Juno | Marriage, women, childbirth — queen of the gods |
| Minerva | Wisdom, strategy, crafts, medicine |
| Mars | War, agriculture, guardian of Rome |
| Venus | Love, beauty, desire, prosperity |
| Mercury | Commerce, travel, communication, thieves |
| Saturn | Time, agriculture, wealth, liberation |
| Diana | Hunt, moon, wilderness, childbirth |
| Vulcan | Fire, forge, craftsmanship |
| Bacchus | Wine, festivity, theatre, freedom |

### Aventine
*The hill of the plebeians, sacred to Diana and Hercules. Associated with the common people.*

| Name | Domain |
|---|---|
| Hercules | Strength, labor, heroism — greatest of Roman heroes |
| Remus | Twin brother of Romulus, co-founder of Rome |
| Ops | Abundance, wealth, the harvest |
| Pax | Peace, goddess of the Pax Romana |
| Spes | Hope, expectation, promise |
| Fides | Faith, honesty, trust between Romans |
| Honos | Honor, military distinction |

### Capitoline
*The most sacred hill, home of the Capitoline Triad: Jupiter, Juno, and Minerva.*

| Name | Domain |
|---|---|
| Roma | Personification of the city and Roman state |
| Victoria | Victory in war and competition |
| Felicitas | Happiness, good luck, success |
| Concordia | Harmony, agreement, civic unity |
| Libertas | Freedom, liberty, manumission |
| Salus | Safety, welfare, public health |
| Pietas | Duty, devotion, loyalty to gods, family, and state |

### Quirinal
*Hill of Quirinus, the deified Romulus. One of Rome's oldest sacred sites.*

| Name | Domain |
|---|---|
| Quirinus | Deified Romulus, god of the Roman people and state |
| Flora | Flowers, spring, fertility of crops |
| Consus | Grain storage, underground granaries, horse racing |
| Furrina | Ancient goddess of springs and wells |
| Carmenta | Prophecy, childbirth, inventor of the Latin alphabet |
| Pales | Flocks, shepherds, the pastoral life |

### Esquiline
*Largest of the seven hills, associated with dawn, magic, and the ancient groves of Rome.*

| Name | Domain |
|---|---|
| Aurora | Dawn, the rosy morning light |
| Lucifer | Light-bringer, the morning star (Venus at dawn) |
| Vesper | Evening star, Venus at dusk |

| Mefitis | Volcanic vapors, sulfurous springs |
| Angerona | Silence, secrecy, the winter solstice |
| Libitina | Death, funerals, the undertakers' guild |

### Viminal
*Smallest of the seven hills, associated with fortune and the quieter household gods.*

| Name | Domain |
|---|---|
| Fortuna | Luck, fate, fortune — the wheel ever turns |
| Genius | The divine spirit within every person and place |
| Janus | Transitions, doorways, beginnings and endings — two-faced |
| Silvanus | Forests, wild fields, uncultivated land |
| Faunus | Forests, secrets, prophecy through dreams |

### Caelian
*The southernmost hill, associated with sky deities and the heavens above.*

| Name | Domain |
|---|---|
| Caelus | The sky itself, Roman equivalent of Uranus |
| Sol | The sun, light, the solar day |
| Luna | The moon, months, the tides |
| Tellus | The fertile earth, earthquakes, marriage |
| Pluto | The underworld, hidden wealth beneath the earth |
| Proserpina | Underworld and spring, daughter of Ceres |
| Ceres | Grain, agriculture, the harvest, motherly love |

## Egyptian — cloud / remote instances

Egyptian mythology is structured around cosmic order (Ma'at) and the journey of the sun through
the underworld each night. Well-suited for remote and cloud systems.

### Aaru
*The Field of Reeds — Egyptian paradise, a perfect mirror of the Nile delta in the afterlife.*

| Name | Domain |
|---|---|
| Osiris | Death, resurrection, the afterlife, fertility |
| Ra | Sun, creation, kingship — the supreme god |
| Horus | Sky, kingship, the falcon-headed son of Osiris |
| Thoth | Writing, wisdom, the moon, scribes and magic |
| Anubis | Death, embalming, the weighing of the heart |
| Isis | Magic, motherhood, healing, the throne |
| Nephthys | Death, service, night, lamentation |
| Hathor | Love, music, beauty, motherhood, the sky |
| Bastet | Cats, home, fertility, protection |
| Sekhmet | War, healing, the desert sun, divine wrath |

### Duat
*The Egyptian underworld through which Ra travels each night, filled with gates and guardians.*

| Name | Domain |
|---|---|
| Ammit | Devourer of hearts deemed unworthy — part lion, hippo, crocodile |
| Sobek | Crocodiles, water, military power |
| Set | Chaos, storms, the desert, foreigners |
| Apep | Chaos serpent, eternal enemy of Ra |
| Khepri | The rising sun, renewal, the scarab beetle |
| Aten | The solar disc, pure light, worshipped by Akhenaten |
| Atum | The complete one, the setting sun, first god of Heliopolis |
| Khnum | Creation, the Nile, potter of human souls |
| Ptah | Craftsmanship, creation through thought and speech |
| Montu | War, the scorching heat of the sun |

### Nun

| Name | Domain |
|---|---|
| Naunet | Female aspect of Nun, the primordial deep |
| Amun | Hidden one, king of the gods, the breath of life |
| Amunet | Female hidden one, consort of Amun |

| Shu | Air, wind, the space between earth and sky |

### Iunu

| Name | Domain |
|---|---|

## Mesopotamian — network devices

The oldest written mythology. Sumerian and Akkadian gods govern the fundamental forces of the world.
Well-suited for infrastructure that underpins everything else.
